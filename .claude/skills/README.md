# Skills du projet (`.claude/skills/`)

Skills [Claude Code](https://docs.claude.com/en/docs/claude-code) versionnés dans le dépôt :
toute personne qui clone TEMPO et travaille avec Claude Code y a accès automatiquement.
Ils portent surtout sur la **qualité de l'UI**, l'**animation** et le **design front** —
pertinents pour une app React comme TEMPO.
